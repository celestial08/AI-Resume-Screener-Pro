# AI Resume Screener Pro

AI-powered Resume Screening System built using Python, Streamlit, SQLite, and NLP.

## Features

* ATS Score Analysis
* Resume Strength Analysis
* Candidate Ranking
* Missing Skills Detection
* Skill Match Percentage
* Email Extraction
* Phone Number Extraction
* Interview Question Generator
* CSV Export
* Analytics Dashboard

## Technologies Used

* Python
* Streamlit
* SQLite
* Pandas
* PyPDF2
* Regex

## How It Works

1. Upload Resume PDF
2. Extract Resume Text
3. Analyze Skills
4. Calculate ATS Score
5. Detect Missing Skills
6. Rank Candidates
7. Generate Interview Questions
8. Provide Smart Recommendations
