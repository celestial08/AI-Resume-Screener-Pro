# AI Resume Screener Pro

AI-powered Resume Screening System built using Python, Streamlit, SQLite, and NLP.

## Features

* ATS Score Analysis
* Resume Strength Analysis
* Candidate Ranking
* Missing Skills Detection
* Skill Match Percentage
* Email & Phone Extraction
* Interview Question Generation
* CSV Export
* Analytics Dashboard

## Technologies Used

* Python
* Streamlit
* SQLite
* Pandas
* PyPDF2
